# Weather App

responsive, web app in react that shows the weather of some city. 
The user should be able to search for a city and save it to favorites

## API
AccuWeather
### Endpoints:
location autocomplete
current weather
5-day daily forecast

### 3rd party libraries
Material Ui
Axios
Moment
react-simple-toasts
validator
