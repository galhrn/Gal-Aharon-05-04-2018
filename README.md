# Weather App

Responsive web app in react that shows the weather of some city. 
The user able to search for a city and save it to favorites.

## API
AccuWeather
### Endpoints:
location autocomplete
current weather
5-day daily forecast

### Features:
Dark-mode
AutoComplete Search

### 3rd party libraries
Material Ui
Axios
Moment
react-simple-toasts
validator
